# My Calculator App

Webde çalışan basit bir hesap makinesi uygulaması.

## Kurulum

1. Projeyi indirin
2. Terminal açın ve proje klasörüne gidin
3. Aşağıdaki komutu çalıştırın:

```bash
npm install
```

4. Uygulamayı başlatmak için:

```bash
npm start
```

Uygulama `http://localhost:3000` adresinde açılacaktır.
